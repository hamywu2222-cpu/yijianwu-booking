## 接下來你只要做這件事（最重要）

### 1. 準備照片

從 `Desktop/一間屋/` 資料夾挑選你最漂亮的照片，複製到：

```
Desktop/yijianwu-website/public/images/
```

建議照片命名（直接複製後改名）：

- `hero.jpg`         ← 最好看的外觀或整體照（用在首頁大圖）
- `room-double.jpg`  ← 雙人房室內
- `room-family.jpg`  ← 家庭房或另一間房
- `room1.jpg` ~ `room8.jpg` 或用 gallery 裡的檔名

### 2. 快速上傳照片

最簡單方式：
1. 開啟 `public/images` 資料夾
2. 把照片拖進去
3. 重新命名成上面建議的檔名

### 3. 測試

```bash
cd Desktop\yijianwu-website
npm run dev
```

開啟 http://localhost:3000 看效果
